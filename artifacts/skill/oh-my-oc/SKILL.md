# oh-my-oc

Use this skill to understand and install the `oh-my-oc` configuration bundle.

## What it does

- `oh-my-oc` is a small CLI for patching an Opencode config directory.
- It applies the managed `opencode.json` and `agent/*.md` files from the release payload.
- It does not install the CLI binary.

## Install

Download `skill.zip` from the matching `oh-my-oc` release, then extract it into `~/.agents/skills` so the final path is `~/.agents/skills/oh-my-oc/`.
